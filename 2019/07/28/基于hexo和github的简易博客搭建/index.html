基于hexo和github的简易博客搭建
发表于2019-07-28|更新于2020-09-08|hexo
|字数总计:675|阅读时长:2分钟|阅读量:|评论数:

环境搭建

一、需要下载 nodejs 环境、 Git 工具  
    nodejs 和 Git 官网下载即可，然后利用终端（ Git Bash、命令行等）进行操作  
二、安装 hexo 博客框架  
    1.安装时借助淘宝镜像源安装 cnpm  
            npm install -g cnpm --registry=https://registry.npm.taobao.org  
            等待安装完成（ cnpm -v 可查看安装的 cnpm 的版本）  
    2.安装 hexo  
        执行命令： cnpm install -g hexo-cli  
        等待安装完成( hexo -v 可用于验证安装是否成功)

创建博客文章

tip：所有的文件操作，可直接使用图像化界面
1.建立空的文件夹目录（记为 blog )，可以使用建立文件夹命令： mkdir blog  
2.进入 blog 目录下，以管理员权限 
    linux 下输入： sudo hexo init  
    windows 下输入：  hexo init  
    等待安装结束，自动生成一下文件  
3.验证框架是否搭建完成  
    启动 hexo: hexo s  或hexo start 
    然后在 localhost：4000 下查看（默认为4000端口）  
4.写文章  
    hexo n "文章名"  或hexo new "文章名"  
    eg: hexo n "myfirstblog"  
    会在 source /_posts 目录下生成 myfirstblog.md  
    进入后启动 vim 进行编辑： vim 文件名
5.最终生成生成   
    回到 blog 下  
    hexo clean 清理界面
    hexo g 或 hexo generate 最后生成

部署到github

1.到 github 新建一个和 github 名一致的仓库名
eg: github 名为 helloworld,则仓库名为 helloworld.github.io
2.回到 blog 目录下安装 git 插件
    执行命令： cnpm install --save hexo-deployer-git
3.相关配置
进入 _config.yml 内的底部配置，如下：
    deploy:
        type: git
        repo: https://github.com/guiyiqiao/guiyiqiao.github.io.git仓库名
        branch: master  
    (据说冒号后没有空格会出错)
4.部署到 github
    执行： hexo deploy   （第一次会要求输入账号 密码）
    部署成功后可前往 github 查看自己的仓库是否有生生成的项目
    查看博客在 仓库名 网址查看  
    eg:  https://helloworld.github.io  

自此，hexo 上传到 github 完成，重要命令 ：
hexo -g   
hexo -d

修改博客的主题（附）

一个 hexo 样式主题的 github 项目 
https://github.com/litten/hexo-theme-yilia
回到 blog 目录下，进行克隆，博客主题放置于 themes 目录下
克隆命令： git clone https://github.com./litten/hexo-theme-yilia.git themes/yilia    

这里可能会出现下载到一部分，显示失败
error: RPC failed; curl 18 transfer closed with outstanding read data remaining
fatal: The remote end hung up unexpectedly
fatal: early EOF
fatal: index-pack failed  
解决方法：
git config --global http.postBuffer 524288000
克隆结束后，在 blog 目录下的 _config.yml 接近底部

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme:landscape 

landscape 修改为 yilia 后重新上传
theme: yilia  (你的主题在 themes 下的名称，是一个文件夹)
